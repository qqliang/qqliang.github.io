关于我的博客

搭建博客的初心

搭建博客的初始原因是完成《开源软件通识》课程作业，通过这次作业了解到了使用 Hexo + Git + Markdown 完成静态博客网站搭建的过程。在这之前，自己虽常浏览别人的博客，但从未想过搭建一个专属自己的博客网站，这次作业正好给了我一次锻炼的机会，让自己思考有了博客之后如何建设它。

平常自己习惯用飞书知识库来整理自己的学习内容，但从未将其公开，它更像是学习过程的记录，而不完全是输出内容。随着在课堂上对开源社区构建了解的深入，我想是时候让自己在公开场合输出内容，以提升自己的思考能力了！

博客主题定位

既然搭建博客的初始原因是来自专业课程，那么就先将博客的初始主题定位为专业技术，平时整理一些学习实践过程中的踩坑经验吧！也希望随着时间的推移，该博客能有更丰富的主题，做到包括但不限于阅读分享、生活思考等！
